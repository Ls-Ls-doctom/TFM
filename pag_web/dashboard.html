Dashboard

Explora en un único lugar indicadores públicos de empleo, renta, desigualdad, población y movilidad. La plataforma reúne datos que originalmente se publican de forma separada por organismos estatales y portales municipales.

forum Consultar en lenguaje natural
Tipos de indicador distintos -- Variables únicas: empleo, renta, IPC, alquiler, turismo…
Ciudades cubiertas -- Barcelona, Madrid, Valencia, Sevilla, Bilbao, Málaga y Zaragoza.
Fuentes de datos -- INE, SEPE y portales de datos abiertos municipales.
Observaciones almacenadas -- Filas individuales en la base de datos (una ciudad, un indicador, un periodo).
