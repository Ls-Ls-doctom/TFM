Este cuestionario sirve para comprobar si el proyecto tiene sentido como solución: si el problema está bien definido, si la web aporta valor real, si el dato es defendible y si el resultado será útil para un usuario final y para la defensa del TFM. Marca cada pregunta con `Sí`, `Parcial` o `No`.

Cómo usarlo

Usa este formulario al inicio del proyecto, después del MVP local y antes del cierre final. Si muchas respuestas quedan en `No` o `Parcial`, eso indica que el proyecto aún no está atacando bien el problema original o que la narrativa necesita ajuste.
